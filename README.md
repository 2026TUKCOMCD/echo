# echo
회상 요법을 통한 치매 예방 시스템
## 개요
본 프로젝트는 경도인지장애를 가진 분들이 일상 속에서 자연스럽게 기억을 회상하고, 긍정적인 감정을 유지할 수 있도록 돕는 AI 대화 시스템입니다.
사용자의 건강 데이터, 위치 정보, 개인 선호도를 활용하여 맞춤형 질문을 생성하고, 하루의 대화를 일기 형식으로 요약하여 기록합니다.
